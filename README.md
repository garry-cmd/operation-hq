# Operation HQ

Personal mission control — Annual Roadmap, Quarterly OKRs, Weekly Plans, and Check-ins.

## Stack
- Next.js 16 (App Router)
- React 19 + TypeScript 5
- Tailwind CSS v4
- Supabase (auth + database)
- Deployed on Vercel

## Dev
```bash
npm install
npm run dev
```

## Env vars
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```
